1. 삼성생명·교보생명·NH농협생명은 다음달 보장성보험의 예정이율을 2.25%에서 2.0%로 조정할 계획이다. 동양생명도 갱신형 보장성 상품과 종신보험에 대한 예정이율을 2.25%로 내릴 예정이다.
2. 업계는 예정이율이 0.25%포인트 떨어지면 신규 또는 갱신 보험계약의 보험료가 평균 7∼13% 오를 것으로 내다보고 있다.
3. 자동차 보험료도 오른다. MG손해보험은 이번달 개인용 자동차보험료를 평균 2% 인상했다.
4. MG손보는 지난해 107.7%의 손해율을 기록했다. 이에따라 내달 손보사들의 자동차 보험료 인상이 불가피할 것이란 관측이 나온다.
5. 실제 캐롯손해보험은 현재 5% 가량 인상안을 논의 중이며, AXA손해보험도 영업용 차량 보험료 인상을 검토 중인 것으로 알려졌다.

const tabBtn = $("#tab-btn > ul > li");   //각각의 버튼을 변수에 저장
const tabCont = $("#tab-cont > div");     //각각의 콘텐츠를 변수에 저장

tabBtn.click(function(){
	let target = $(this);
	let index = target.index();
	
	tabBtn.removeClass("active");
	target.addClass("active");
	tabCont.css("display","none");
	tabCont.eq(index).css("display","block");
});
