사이트의 레이아웃은 어떻게 작업할까요?

학습 주제

CSS 그리드 레이아웃(Grid Layout)은 페이지를 여러 주요 영역으로 나누거나 크기와 위치 및 문서 계층 구조의 관점에서 HTML 기본요소를 작성된 요소간의 관계를 정의하는데 아주 탁월한 기능을한다.
